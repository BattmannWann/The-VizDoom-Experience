# Example Files Directory

This directory contains hand-crafted logic to run ViZDoom, calculating rewards received from agent performance. No training takes places in these files as the agent is set to just make a random action. These files serve as a simple example for building foundational knowledge on the inner workings of ViZDoom and for simple local testing before transferring logic into an actual training instance. 

One file calculates the rewards manually and the other obtains the rewards generated by the ACS script underneath the WAD file.
